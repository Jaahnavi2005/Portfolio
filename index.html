01 · About

03 · Featured projects
TEAM PROJECT · 2 OrbitWatch
I built the frontend · my teammate built the backend

A real-time space debris tracking platform, submitted to Space Hackpad 2026. It pulls live orbital data and renders object positions on an interactive 3D globe, so debris that's normally just a row in a catalog becomes something you can actually see moving.

I built the entire frontend — the CesiumJS globe rendering, the UI layer, and how orbital paths get visualized in 3D space — while my teammate handled the backend, including pulling and processing live CelesTrak data through the SGP4 propagation model.

My contribution
CesiumJSFrontend UI3D visualization
Problem solved

Made orbital debris density visible and explorable in real time, instead of buried in a static catalog.

SOLO PROJECT · IN PROGRESS SheRise
Designing and building solo · work in progress

A three-module React app addressing UN SDGs 3, 4, and 5 — health, education, and gender equality. Each module tackles a different gap: MindSpace for mental wellbeing, BodyWise for health literacy, and CareerSpark for career guidance, all powered by the Claude API.

I'm designing and building the whole thing solo — from the React component architecture and routing to integrating the Claude API across all three modules. Still a work in progress, with modules being built out one at a time.
